## Direct Competitors

### 1. Hgraph (hgraph.com)

**What they do:**
- Hedera Stats â€” open-source Grafana dashboards
- GraphQL API for mirror node data
- Metrics: network revenue, TVL, stablecoins, active accounts, time-to-consensus

**Strengths:**
- Open-source methodologies (transparency)
- Enterprise-grade infrastructure (bare-metal mirror nodes)
- Developer-focused (GraphQL API)
- Backed by known Hedera community figures

**Weaknesses:**
- More infra/enterprise focused than retail
- Dashboards require Grafana knowledge
- No social/community features

**Differentiation opportunity:**
- OnlyFlies: Token-specific analytics + social integration
- More accessible UI for retail users
- Alert systems (whale monitors, price alerts)
- Agent coordination layer (ClawSwarm)

### 2. HashScan (hashscan.io)

**What they do:**
- Official Hedera explorer
- Transaction/account/token lookup
- Real-time network activity

**Strengths:**
- Official, trusted
- Clean UI
- Comprehensive data

**Weaknesses:**
- No analytics layer
- No portfolio tracking
- No alerts/notifications

### 3. CoinStats (coinstats.app/explorer/hedera)

**What they do:**
- Multi-chain portfolio tracker
- Hedera wallet viewing
- NFT gallery

**Strengths:**
- Multi-chain (users already there for BTC/ETH)
- Mobile apps
- Portfolio aggregation

**Weaknesses:**
- Hedera is afterthought, not focus
- No Hedera-specific insights
- No DeFi/DEX analytics

### 4. Hedera Explorer by Acoer

**What they do:**
- Healthcare-focused blockchain explorer
- Enterprise use cases

**Strengths:**
- Niche focus (healthcare)
- Enterprise relationships

**Weaknesses:**
- Very narrow focus
- Not relevant to retail/DeFi

## Strategic Recommendations

### 1. Double down on token-specific analytics
- Individual token dashboards (volume, holders, whale movements)
- MemeJob token tracking (launches, ascensions)
- Historical price/volume charts

### 2. Build the best whale monitoring
- Real-time alerts via Telegram/Discord
- Configurable thresholds per token
- Verified data (Mirror Node, not scraped)

### 3. Integrate with ClawSwarm
- Agents can query OnlyFlies for data
- Task bounties for maintaining analytics
- Governance for adding new token coverage

### 4. Social/Community layer
- Token holder leaderboards
- Community sentiment tracking
- Content curation (news, analysis)

### 5. Freemium model
- Free: Basic token lookup, whale alerts for top tokens
- Premium: Custom alerts, API access, portfolio tracking
